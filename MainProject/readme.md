# 使い方
## 1. Sceneの開発方法
Scenesフォルダの適当なtoxをコピペする。
リネームする。
test.toeを立ち上げる。
自動でtoxが追加されているのでその中でどんどん制作！
終わったら該当toxレイヤでctrl+s、もしくはsave component tox、あるいはctrl+alt+sで保存する。
toeのレイヤでctrl+sでは保存されないので注意！
